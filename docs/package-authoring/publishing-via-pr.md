# Publishing via PR

How to publish packages to the Kexra ecosystem.

## Why this exists

Pull request publishing ensures quality and maintains the package registry.

## How it works

Fork the packages repository, add your package, and submit a pull request for review.

## Examples

1. Fork kexra-packages repo
2. Add package to packages/your-package/
3. Add metadata to site/packages/your-package.json
4. Submit PR with description

## Common mistakes

- Not testing package locally first
- Incomplete metadata
- Not following review feedback
