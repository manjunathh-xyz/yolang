# Kex Fmt

Format Kexra source files.

## Why this exists

Consistent code formatting improves readability and maintainability.

## How it works

`kex fmt` reformats source files according to Kexra's style guidelines.

## Examples

```bash
# Format all .kx files in current directory
kex fmt

# Format specific file
kex fmt my-file.kx

# Format directory recursively
kex fmt src/
```

## Common mistakes

- Formatting files with syntax errors
- Not committing formatted changes
- Overriding manual formatting decisions
