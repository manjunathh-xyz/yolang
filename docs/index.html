A small, friendly scripting language built for learning. v0.4.1

Welcome to Kexra

Kexra is a simple interpreted scripting language designed for beginners, hobbyists, and anyone interested in language design. It's built with TypeScript and runs on Node.js.

say "Hello, Kexra"

set x = 5

check x > 3 {
  say "x is big"
}

fn add(a, b) {
  return a + b
}

say add(2, 3)

Install & Run

npm install -g kexra
kex run examples/hello.kx

Documentation

Syntax Overview

# Variables
set name = "Kexra"
set age = 25

# Output
say "Hello World"
say name + " is " + age + " years old"

# Math
set result = (10 + 5) * 2 / 3

# Conditionals
check age >= 18 {
  say "Adult"
} else {
  say "Minor"
}

# Loops
set i = 0
loop i < 5 {
  say i
  set i = i + 1
}

# Functions
fn add(a, b) {
  return a + b
}

say add(2, 3)

CLI Commands

Command Description Example
