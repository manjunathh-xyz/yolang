A small, friendly scripting language built for learning.

Welcome to Kexra

Kexra is a simple interpreted scripting language designed for beginners, hobbyists, and anyone interested in language design. It's built with TypeScript and runs on Node.js.

say "Hello, Kexra"

set x = 5

check x > 3 {
  say "x is big"
}

fn add(a, b) {
  return a + b
}

say add(2, 3)

Install & Run

npm install -g kexra
kex run examples/hello.kx
