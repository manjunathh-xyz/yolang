# Troubleshooting

Common issues and their solutions when getting started with Kexra.

## Why this exists

Setup problems can discourage beginners. This page provides quick fixes for frequent issues.

## How it works

Check symptoms against solutions, following step-by-step fixes.

## Examples

- Command not found: Reinstall globally or check PATH
- Syntax errors: Verify quotes and newlines

## Common mistakes

- Not reading error messages carefully
- Assuming installation issues are bugs
