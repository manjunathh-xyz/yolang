# Learning Path

A suggested progression for learning Kexra, from basics to advanced topics.

## Why this exists

A structured learning path helps beginners build knowledge incrementally without feeling overwhelmed.

## How it works

Start with installation and hello world, then progress through language features in logical order.

## Examples

1. Installation
2. Hello World
3. Variables and types
4. Functions
5. Control flow
6. Modules

## Common mistakes

- Jumping to advanced topics too early
- Skipping fundamentals
