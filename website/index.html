Kexra

A simple, educational scripting language

Learn Programming Through Simplicity

Kexra is an interpreted scripting language designed for beginners, students, and anyone interested in programming language design. Built with TypeScript and running on Node.js.

say "Hello, Kexra"

set x = 5

check x > 3 {
  say "x is big"
}

fn add(a, b) {
  return a + b
}

say add(2, 3)

Install & Run

npm install -g kexra
kex run examples/hello.kx
