# Number Operations

Numbers support arithmetic operations for mathematical computations.

## Why this exists

Number operations enable calculations and data processing.

## How it works

Standard arithmetic operators work on numbers: +, -, \*, /, %.

## Examples

```kx
set a = 10
set b = 3
say a + b  # 13
say a / b  # 3.333...
```

## Common mistakes

- Division by zero
- Floating-point precision issues
